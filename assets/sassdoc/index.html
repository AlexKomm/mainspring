    ru

    @function ru($rhythm-units: 1) { ... }

    Description

    Simple vertical rhythm function returns a REM unit that is a fraction of the line-height of the body. This is useful for maitaining a vertical rhythm in your layouts. Typically using round fractions or mutlipliers (1/4, 1/2, 1, 1.5, 2).

    "RU" is short hand for "Rhythm Unit"

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $rhythm-units

    units of "leading" defined by default line-height

    		Integer1

    Example

    Basic Usage Sass

    .my-module {
  padding: ru(0.5) ru(1.25);
}

    Basic Usage CSS Output

    .my-module {
  padding: 0.75rem 1.875rem; // 12px / 30px
}

Assuming these values
$base-line-height: 24px
$base-font-size: 16px

    Requires

    • [function] rem
    • [variable] base-line-height

    $svg-pad-left
